Statistics & Inference
Where probability meets data. The toolkit for making confident, honest claims from noisy observations — the foundation of A/B testing and scientific ML.
p-value — the most misunderstood number in science
P(observing data this extreme | H₀ is true). NOT the probability that H₀ is true.
⚠️ Three Things p-value Is NOT

1) Not the probability H₀ is true. 2) Not the probability your result happened by chance. 3) Not a measure of effect size. With 10M users, p < 0.001 for a 0.0001% CTR change — statistically significant, practically irrelevant. Always pair with effect size + CI.

📖 The Coin Story

You flip a coin 10 times, get 9 heads. p ≈ 1% = "if this were a fair coin, I'd see this by luck only 1% of the time." Since 1% < 5% (your threshold), you reject "it's a fair coin." p = 30%? Meh — totally plausible by chance, don't reject.

Type I Error (α) — False Positive
Reject H₀ when it's true. See an effect that doesn't exist.
🔥 Fire alarm, no fire

You ran outside for nothing. α = 0.05 means 5% chance of this mistake per test.

Type II Error (β) — False Negative
Fail to reject H₀ when it's false. Miss a real effect.
🏠 Real fire, no alarm

Power = 1 − β = probability of detecting a real effect. More data = higher power.

Confidence Interval & Hypothesis Testing
$$\begin{aligned} 95\%\ \text{CI:} \quad \bar{x} \pm 1.96 \times \frac{\sigma}{\sqrt{n}} \end{aligned}$$
5-step process: 1. H₀ (no effect) vs H₁ (effect exists) 2. Choose α (usually 0.05) 3. Collect data, pick test 4. Compute p-value 5. p < α → reject H₀ (≠ accept H₁!)
💡 CI Interpretation

A 95% CI does NOT mean "95% probability the true value is in this range." It means: if we repeated this experiment 100 times, ~95 of those intervals would contain the true value. The interval either contains the true value or it doesn't.

🎯 Interview Gold

"I always pair p-values with effect sizes and confidence intervals. For A/B tests I pre-register sample size (power analysis) before peeking — peeking inflates Type I error. At massive scale, almost everything becomes 'significant' — the real question is whether the effect size matters for the business. For skewed metrics like revenue, I prefer Mann-Whitney U or bootstrap CIs over parametric tests."
