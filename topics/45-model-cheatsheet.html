Data Problem Diagnostic Checklist
  • ✅ Class imbalance? → PR-AUC metric, class weights or SMOTE, threshold tuning
  • ✅ High cardinality categoricals? → target encoding with CV, embeddings, count encoding
  • ✅ Missing data? → add missingness indicator + impute; XGBoost handles NaN natively
  • ✅ Multicollinearity? → check VIF, use Ridge or PCA before linear models
  • ✅ Outliers? → Huber loss for regression; robust scalers; Winsorize at 1st/99th percentile
  • ✅ Data leakage? → features collected after prediction time; always check temporal ordering
  • ✅ Label noise? → label smoothing, robust loss functions, confidence learning (cleanlab)
  • ✅ Distribution shift at serving? → PSI monitoring, trigger retraining when PSI > 0.1
