Perceptron & Multi-Layer Perceptron
From the simplest possible model (a weighted sum) to the universal function approximator that powers modern AI — the difference is just layers and non-linearities.
Perceptron — The Atom
$$\begin{aligned} z = w_1x_1 + w_2x_2 + ... + w_nx_n + b = w^Tx + b \\ output = activation(z) \end{aligned}$$
Learned by updating weights toward correct outputs (perceptron learning rule). Can only learn linearly separable patterns — hence the need to stack layers.
Multi-Layer Perceptron — forward pass signal flow
Forward Pass — Layer by Layer
$$\begin{aligned} h^1 = activation(W^1x + b^1) \\ h^2 = activation(W^2h^1 + b^2) \\ \hat{y} = \text{softmax}(W^3h^2 + b^3) \quad \text{(classification)} \\ \hat{y} = W^3h^2 + b^3 \quad \text{(regression)} \end{aligned}$$
Without activation functions, stacking linear layers collapses to a single linear layer: W₃(W₂(W₁x)) = Wx. The non-linearity is what enables learning non-linear decision boundaries.

A neural network is a brilliant intern. 10M training examples + 6 months = outperforms everyone. 200 examples on day one = memorizes them exactly, fails on anything new. And it can never explain its decisions. "I just... feel it's a cat."

🎯 Interview Gold

"Universal approximation theorem: a single hidden layer with enough neurons can approximate any continuous function on a compact domain. But 'enough neurons' can be exponential — depth is more efficient than width. Each layer learns increasingly abstract representations: pixels → edges → shapes → objects. For tabular data with <100K rows, XGBoost almost always beats an MLP. Neural networks shine when data is unstructured (images, text) or when scale is massive."
